\section{Компиляция и установка}

Для компиляции и установки программы необходимо выполнить следующие действия:

\begin{enumerate}
\item скопировать файл архива \productcodefull\sdash 01\enspace 12\enspace01.zip на \acsu{эвм}, где будет осуществляться сборка;
\item проверить контрольную сумму скопированного архива на соответствие приведенной в \productcodefull\sdash 01\enspace 12\enspace 01\sdash УЛ, используя штатные средства операционной системы;

\item распаковать архив в рабочую папку, используя штатные средства операционной системы;

\item из директории с распакованными файлами выполнить команду сборки в терминале:

\lstinline|mkdir build && cd build && cmake .. && make && cpack|

и дождаться окончания сборки;

\item после окончания сборки в директории \lstinline|/build| появится собранный deb пакет.

\item произвести установку собранного пакета на требуемых \acsu{эвм} от имени суперпользователя используя команду \lstinline|sudo apt install|;

\item в случае возникновения ошибок при сборке/установке из-за отсутствующих зависимостей, удовлетворить данные зависимости путем установки требуемых библиотек и повторить процесс сборки/установки.
\end{enumerate}
